MEU PRIMEIRO PORTFÓLIO

1. Começar criando um CRUD básico com as seguintes funções:

Criar usuário,
Logar usuário,
Listar usuário,
Deletar usuário.

Usando as seguintes rotas:

/api/v1/register
/api/v1/login
/api/v1/update/user/:id
/api/v1/list/user/:id
/api/v1/delete/user/:id
